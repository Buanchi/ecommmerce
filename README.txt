## Ecommerce SEIN

Ecommerce de ropa responsive hecho con Javascript vanilla, Boostrap, SweetAlert2, Toastify y la api "products JSON"

## Descripcion

Listado de productos, carrito, botones para agregar o restar unidades y boton para eliminar todos los productos 
Informacion del carrito guardada y cargada desde el local storage
Boton de check out que redirecciona a una experiencia de pago con formularios de shipping y payment
